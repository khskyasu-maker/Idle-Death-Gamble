- 제외: パチスロ(슬롯), 4円パチ 전용, 저대여 미확인 기종, 이름이 정확히 맞지 않는 변형 모델

- 이 표는 DMM 공식 1円 순위가 아닙니다.
- PV 인기순은 DMM 페이지 조회/관심도 계열이며 실제 착석률, 가동률, 점포별 사용 수가 아닙니다.
- DMM의 PV 집계 기간과 보정 산식은 공개 설명이 제한적입니다.
- 이 문서는 방문 지시, 당첨 예측, 수익 보장, 추천 점수가 아닙니다.
